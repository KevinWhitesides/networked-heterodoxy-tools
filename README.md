# Networked Heterodoxy Tools

This repository contains computational tools for analyzing recurring discourse patterns and cultural transmission networks across structured datasets.

The tools support analysis of:

- binary incidence matrices (case × trope or case × feature datasets)
- co-occurrence relationships among cultural elements
- bipartite and projected networks
- topic-derived networks from textual corpora

The project grew out of the dissertation:

**Networks of Heterodoxy: Shared Dissent and the Dynamics of Counter-Discourse**

The repository is organized into modules corresponding to different research pipelines and analytic tools.

---

## Dissertation Resources

Scripts, workflows, and supporting materials related to the dissertation are located in:

/dissertation

This includes tools for:

- co-occurrence diagnostics
- bipartite and projected network construction
- network metric analysis
- topic modeling workflows
- reproducible analysis pipelines

Example datasets and workflow documentation are included to allow others to reproduce and experiment with the methods.

---

## Project Scope

While the initial materials derive from the dissertation, the repository is intended to evolve into a broader toolkit for studying **networked heterodoxy** and related forms of cultural and discursive transmission.

Future additions may include:

- generalized network analysis utilities
- improved corpus processing tools
- visualization workflows
- additional case studies and datasets
